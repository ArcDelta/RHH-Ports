## Installation
Copy your Triga game files to `triga/assets`. If using Steam data, the game will be patched to stretch to fit.

## Default Gameplay Controls
| Button            | Action |
|--                 |--|
| SELECT            | Quit to menu |
| D-PAD / JOYSTICK  | Move cursor |
| A                 | Select / place piece |
| R1                | Reset level |
| L1                | Undo |
