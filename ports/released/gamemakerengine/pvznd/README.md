## Installation
Download the *Windows* version from https://crisdevelop.itch.io/pvznd and add the zip to the `pvznd/assets` folder.

## Default Gameplay Controls
| Button            | Action |
|--                 |--|
| START             | Pause |
| D-PAD / JOYSTICK  | Move cursor |
| A                 | Left click |
| B                 | Right click |
