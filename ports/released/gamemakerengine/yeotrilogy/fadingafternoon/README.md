## Installation
Copy your gamew files to `fadingafternoon/assets`.

## Default Gameplay Controls (Xbox Layout)
| Button            | Action |
|--                 |--|
| START             | Pause |
| SELECT            | Quit to Title |
| D-PAD / JOYSTICK  | Move |
| X                 | Punch / Accept |
| Y                 | Kick / Cancel |
| L1                | Hints |
| R1                | Item |
