## Installation
Copy all game data to `outridermako/assets`. Requires GMToolkit runtime.

## Default Gameplay Controls (Native Gamepad)
| Button            | Action |
|--                 |--|
| START             | Pause |
| JOYSTICK          | Move / Aim |
| D-PAD LEFT        | Book |
| D-PAD RIGHT       | Inventory |
| A                 | Jump |
| B                 | Secondary Attack |
| X                 | Main Attack |
| Y                 | Interact |
| L1                | Prev Tab |
| R1                | Next Tab |
| R2                | Use Item |

## Default Gameplay Controls (No Analog)
| Button            | Action |
|--                 |--|

| D-PAD             | Move / Aim (WASD) |
| A                 | Jump |
| B                 | Secondary Attack |
| X                 | Main Attack |
| Y                 | Interact |
| L1                | Book |
