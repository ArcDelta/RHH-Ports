## Installation
Copy your `arrest of a stone buddha.exe` file to `arrestofastonebuddha/assets`.

## Default Gameplay Controls (Xbox Layout)
| Button            | Action |
|--                 |--|
| START             | Pause |
| SELECT            | Quit to Title |
| D-PAD / JOYSTICK  | Move |
| X                 | Shoot / Accept |
| Y                 | Kick / Cancel |
| R1                | Aim |
