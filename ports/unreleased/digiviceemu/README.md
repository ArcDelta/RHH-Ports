## Install
Down load the Windows zip from https://k0as7.itch.io/digivice-emulator-d3-v10-eu and unzip to the assets folder.

## Default Gameplay Controls
| Button            | Action                                |
|--                 |--                                     |
| A                 | Confirm                               |
| B                 | Cancel                                |
| X                 | Menu                                  |
| Y                 | Reset (Hold)                          |
| L1                | Toggle sounds                         |
| R1                | Toggle CRT                            |
