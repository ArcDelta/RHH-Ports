## Installation
Copy your game data to `friendsofringo/assets` and let the patcher do its thing.

## Default Gameplay Controls
| Button            | Action |
|--                 |--|
| START             | Pause |
| D-PAD / JOYSTICK  | Move |
| A                 | Kick / Smoke / Cancel |
| X                 | Punch / Activate / Accept |
| L1                | Show Menu |
| R1                | Block / Aggr Mode / Book |
