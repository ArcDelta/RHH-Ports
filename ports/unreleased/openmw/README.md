## Install
Copy your `The Elder Scrolls III: Morrowind GOTY Edition` data from the `Data_Files` folder into `openmw/data`.

## Default Controls
| Button | Action|
| -- | -- |
| A | Activate |
| B | Inventory |
| X | Ready weapon |
| Y | Ready magic |
| L1 | Journal |
| L2 | Jump, Toggle Mouse |
| R1 | Rest |
| R2 | Use |
| L3 | Sneak |
| R3 | Toggle POV |
| Guide | Quick save |
